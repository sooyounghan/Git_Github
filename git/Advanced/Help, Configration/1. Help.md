-----
### git help
-----
1. Git 사용 중 모르는 부분이 있을 때 도움을 받을 수 있는 기능
```
git help
```
2. 기본적인 명령어들과 설명

-----
### git help -a
-----
1. Git의 모든 명령어들 확인 가능
2. vim 모드로 진입하므로 j로 내리기, k로 올리기, :q로 닫아야함

-----
### git (명령어) -h
-----
: 해당 명령어의 설명과 옵션을 볼 수 있음

-----
### git (명령어) --help 
-----
```
git (명령어) --help
```
1. 해당 명령어의 설명과 옵션을 웹 사이트에서 볼 수 있음
2. 웹 사이트가 열리지 않으면, -w를 추가로 붙일 것
